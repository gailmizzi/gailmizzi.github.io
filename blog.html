SFX Makeup

SFX Makeup is Makeup with prosthetics created by using Liquid Latex, Spirit Gum, Scar Wax, and using other basic materials like Tissues, Bandages, Aluminium Foil, and Clingfilm. To bring the character to life, we use colour which is mostly body paint, and basic powdered shadows.

Fantasy/Creative

Fantasy/Creative Makeup is created by only using Body Paint and Powdered Shadows as it doesn't really have prosthetics, and it will look like and illusion of transformation. hence why you can tranform yourself/other people into any other human/ fictional character by using shadows.
